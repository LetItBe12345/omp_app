# MVP-05：文件树、预览与 Diff

- 状态：未开始
- 优先级：P1
- 前置任务：MVP-01
- 后续任务：MVP-07

## 目标

提供完成本地代码任务所需的最小文件能力，但不扩张成完整 IDE。

## 明确不做

- 不做调试器、插件市场和完整语言工作台。
- 不在启动时递归扫描整个 Workspace。
- 不实现复杂项目索引。
- 不常驻加载大型编辑器和 Diff 组件。

## 任务清单

### 文件树

- [ ] 右侧上下文面板中显示文件树。
- [ ] 目录展开时再读取子项。
- [ ] 默认遵守 `.gitignore`。
- [ ] 支持刷新和文件系统变更合并。
- [ ] 支持按名称搜索文件。
- [ ] 对超大目录设置数量限制和明确提示。
- [ ] 路径操作限制在当前 Workspace 内。

### 文件预览与编辑

- [ ] 支持打开 UTF-8 文本文件。
- [ ] 支持只读预览大文件，并限制读取大小。
- [ ] CodeMirror 6 按需加载。
- [ ] 支持最小文本编辑和保存。
- [ ] 保存前检测文件是否被外部修改。
- [ ] 二进制、未知编码和无权限文件显示明确状态。
- [ ] 当前打开文件使用顶部标签，不把会话放到顶部标签。

### Git 状态与 Diff

- [ ] 读取当前 Workspace 的 Git 状态。
- [ ] 显示新增、修改、删除和未跟踪文件。
- [ ] 支持查看单文件 Diff。
- [ ] 支持查看工作区总 Diff。
- [ ] Diff 高亮组件按需加载。
- [ ] 大 Diff 使用截断、分页或虚拟化。
- [ ] Diff 可以加入 `@diff` 上下文。

### 与对话联动

- [ ] 文件和目录可以加入 Prompt 上下文。
- [ ] Tool Call 修改文件后刷新对应节点和 Git 状态。
- [ ] Assistant 输出中的文件路径可以打开预览。
- [ ] 文件改动卡片可以跳转到 Diff。
- [ ] 不将完整文件树或完整仓库内容自动发送给模型。

### 测试

- [ ] 测试目录懒加载和 `.gitignore`。
- [ ] 测试路径越界和符号链接边界。
- [ ] 测试文本读取、保存和外部修改冲突。
- [ ] 测试非 Git Workspace。
- [ ] 测试大文件和大 Diff 的限制逻辑。
- [ ] 测试 Tool Call 后的文件状态刷新。

## 完成条件

- [ ] 可以浏览当前 Workspace 文件。
- [ ] 可以打开、简单编辑和保存文本文件。
- [ ] 可以查看 Git 状态和 Diff。
- [ ] 文件树不会在启动时递归扫描完整项目。
- [ ] 大文件、大目录和大 Diff 不会阻塞主线程。
- [ ] 文件和 Diff 可以加入对话上下文。

## 复用重点

- CodeMirror 6：轻量文本编辑。
- OpenCode Session UI：Diff 和文件改动交互。
- 成熟的 `.gitignore` 匹配库：不要手写规则解析器。
- Git CLI：MVP 不引入大型 Git SDK。
